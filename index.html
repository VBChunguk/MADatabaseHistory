MA Card Info

확밀아 카드 정보 변화를 기록!!

Download this project as a .zip file Download this project as a tar.gz file

확밀아 카드정보 뷰어는 안드로이드 애플리케이션으로, 확밀아 카드 데이터베이스를 읽고 카드 정보를 제공합니다.

컴파일된 앱 다운로드

이곳에서 최신 버전을 찾고 다운로드할 수 있습니다.

주요 기능과 특징

  • 카드 데이터베이스 변화 기록: 확밀아 데이터베이스에 변화가 생기면, 그 변화를 감지하고 기록합니다.
  • 카드 정보 보기: 일러스트, 스킬, 설정 등의 자료를 열람할 수 있습니다.
  • 카드 검색: 여러 가지 검색 필터를 적용해 찾고자 하는 카드를 검색할 수 있습니다.
  • 카드 공유: 카드 정보, 일러스트 등의 자료를 다른 사람과 공유할 수 있습니다.
  • 고화질 카드 일러스트: 일러스트 뷰어에서는 고화질의 카드 일러스트를 다운로드해서 보여 줍니다.
  • 쓰기 편한 디자인: 디자인을 단순화하여 기능을 찾기 쉽게 만들었습니다.

장점

카드 데이터베이스 변화 기록 기능이 있어서...

  • 언제 어떤 새로운 카드가 추가되었는지 한눈에 볼 수 있습니다.
  • 카드가 수정되어도 따로 저장된 옛 버전이 남아 있기 때문에, 수정된 정보를 옛 버전과 비교하며 볼 수 있습니다.

카드 검색 기능이 있어서...

  • 빠르게 자신이 찾고자 하는 카드를 찾을 수 있습니다.
  • 여러 가지 검색 필터를 사용해 카드 이름이 기억나지 않아도 원하는 카드를 찾을 수 있습니다.

카드 공유 기능이 있어서...

  • 그 작디작은 액정에서 카드 정보를 번거롭게 복사/붙여넣기할 수고를 덜 수 있습니다.
  • 카드 일러스트를 찾아 나설 필요가 없습니다.
